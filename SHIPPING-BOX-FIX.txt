THE PROBLEM:
The Shipping Center's box grid shows "AI PICK" on the box
that suggestPackage() recommends. But suggestPackage() uses
category-based matching which can be WRONG for the actual
item dimensions.

Example: Leica M6 camera (10x8x6 inches)
— suggestPackage returns boxSize: "small" (camera category)
— Small preset is 10x8x4
— Item is 6" tall, box is only 4" tall
— Fit indicator correctly shows "Too small" in red
— But AI PICK badge is STILL on Small
— The correct box is Large (18x14x12) which shows "Fits"
— The user sees contradictory information: AI says Small
  but the fit check says Too Small

Example: HP Envy 15 laptop (14x9.5x0.7 inches)
— After the thin-item fix, suggestPackage returns "custom"
— Custom box should be highlighted and dims pre-filled
— This may already work — verify

THE FIX:
In the BOX_PRESETS grid rendering (PreSaleTab and
ReadyToShipTab wizard step 1), add a VALIDATION step:

After computing the fit indicator for each box, check
if the AI-suggested box (item.aiBox) has a "Too small"
fit indicator. If it does:

1. Find the FIRST preset in order (tiny→small→medium→
   large→xl→oversized→furniture) that has a "Fits" indicator
2. Show "AI PICK" badge on THAT box instead
3. Show a note: "AI upgraded from [original] to [new]
   based on actual dimensions"

If NO standard preset fits well (all are "Loose" or
the smallest "Fits" box has excessive void):
4. Show "AI PICK" on Custom
5. Pre-fill custom dims: itemL+4, itemW+4, itemH+4
6. Show: "Custom recommended — sized for your item"

ALSO: When Custom IS the AI recommendation (item.aiBox
=== "custom" or autoCustom is true):
7. The Custom card should be highlighted with teal border
   (same as any other AI PICK box)
8. The dim fields should be pre-filled with the AI dims

In the BOX_PRESETS grid (PreSaleTab), BEFORE the
grid renders, compute the VALIDATED AI pick:

  // Validate AI pick against actual fit
  const itemL = Number(editDims.length) || item.aiBoxDims?.length || 0;
  const itemW = Number(editDims.width) || item.aiBoxDims?.width || 0;
  const itemH = Number(editDims.height) || item.aiBoxDims?.height || 0;
  const hasItemDims = itemL > 0 && itemW > 0 && itemH > 0;

  let validatedAiPick = item.aiBox || "";
  let aiPickUpgraded = false;
  let aiPickReason = "";

  if (hasItemDims && validatedAiPick && validatedAiPick !== "custom") {
    const aiPreset = BOX_PRESETS[validatedAiPick];
    if (aiPreset) {
      // Check if AI pick actually fits
      const fitsAi = (itemL + 4) <= aiPreset.l
        && (itemW + 4) <= aiPreset.w
        && (itemH + 4) <= aiPreset.h;

      if (!fitsAi) {
        // AI pick doesn't fit — find the first preset that does
        const presetOrder = ["tiny", "small", "medium", "large", "xl", "oversized", "furniture"];
        let foundFit = false;
        for (const key of presetOrder) {
          const p = BOX_PRESETS[key];
          if ((itemL + 4) <= p.l && (itemW + 4) <= p.w && (itemH + 4) <= p.h) {
            // Check for excessive void
            const maxVoid = Math.max(p.l - (itemL + 4), p.w - (itemW + 4), p.h - (itemH + 4));
            const minItemDim = Math.min(itemL, itemW, itemH);
            if (maxVoid < minItemDim * 3) {
              // Good fit — use this preset
              validatedAiPick = key;
              aiPickUpgraded = true;
              aiPickReason = `Upgraded from ${item.aiBox} (too small) to ${p.label}`;
              foundFit = true;
              break;
            }
          }
        }
        if (!foundFit) {
          // No preset fits well — recommend custom
          validatedAiPick = "custom";
          aiPickUpgraded = true;
          aiPickReason = `Custom box recommended — ${Math.ceil(itemL + 4)}×${Math.ceil(itemW + 4)}×${Math.ceil(itemH + 4)}" fits perfectly`;
        }
      }
    }
  }
