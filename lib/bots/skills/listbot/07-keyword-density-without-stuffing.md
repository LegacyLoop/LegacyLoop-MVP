# Keyword Density Is a Real Thing — And There's a Sweet Spot

Search algorithms on every major platform (eBay, Etsy, Google via eBay listings, Facebook Marketplace) weight keyword density — how often your target keyword appears relative to total word count. Too low and you don't rank. Too high and you get flagged as spam.

The sweet spot is **7-15% density**.

## The Math

Keyword density = (times keyword appears ÷ total word count) × 100

**Example:** A 200-word description that mentions "teak credenza" 4 times has 2% density for that exact phrase. Too low.

**Target:** The same description should mention variations of "teak credenza" (including "teak sideboard," "mid-century credenza," "Danish credenza," "teak cabinet") 15-25 times total for 7-12% density.

## The Natural Variation Rule

DON'T stuff the same exact phrase repeatedly. Use NATURAL variations:

**Bad (stuffed):**
"This teak credenza is a beautiful teak credenza from the teak credenza era. The teak credenza has original hardware. This teak credenza measures 60 inches."

**Good (natural variation):**
"This mid-century teak credenza is a 1958 Danish design classic. The sideboard features original brass hardware and solid teak construction. The cabinet measures 60 inches wide — a versatile size for dining room storage or media console use."

Same keywords (credenza, sideboard, cabinet, teak) but read naturally.

## Semantic Variations to Always Include

For any item, your description should naturally weave in:
- **The primary keyword** (exact match): "teak credenza"
- **Synonym 1** (functional equivalent): "teak sideboard"
- **Synonym 2** (different angle): "teak cabinet," "teak console"
- **Era qualifier variations:** "mid-century," "1950s," "Danish modern," "vintage"
- **Style qualifier variations:** "Scandinavian design," "minimalist furniture"
- **Use case variations:** "dining room storage," "media console," "entryway furniture"

Each natural variation adds keyword weight without triggering spam filters.

## The Red Flag Density Threshold

Keyword density above 20% triggers spam detection on most platforms. Above 25% gets listings auto-flagged for review or removal.

**Never exceed 15% density for any single exact-match phrase.**

## The Platform-Specific Density

Different platforms have slightly different tolerances:
- **eBay:** 7-12% density (algorithm is aggressive on spam)
- **Etsy:** 5-10% density (smaller keyword budgets, tags carry more weight)
- **Facebook Marketplace:** 3-7% density (conversational style, low tolerance for repetition)
- **Craigslist:** 5-10% density (no algorithm, but buyers scroll past spam)
- **1stDibs:** 2-5% density (prose-heavy, elegant tone)

## The Title-to-Description Flow

The keywords you use in the title should ALSO appear in the description — ideally in the first 100 words. This signals to the algorithm that the title isn't clickbait.

**Title:** "Hans Wegner CH-25 Lounge Chair 1958 Original Paper Cord Teak Danish Modern"

**First 100 words of description MUST include:**
- Hans Wegner
- CH-25
- 1958
- Paper cord
- Teak
- Danish Modern (or "Danish modernism")

If the title promises keywords that don't appear in the description body, the algorithm flags it as keyword-stuffing.

## The Long-Tail Keyword Strategy

Long-tail keywords (3-5 word phrases) should appear at least ONCE in both title and description:
- "mid century modern teak credenza"
- "Hans Wegner CH-25 paper cord"
- "vintage Danish modern lounge chair"

Long-tails convert better than short-tails because buyers searching them are already committed.

## Output Format

When generating a description, internally track keyword density as you write. Aim for:
- Primary keyword: 7-10% density
- Variations (synonyms, semantic): 5-8% density
- Total keyword coverage: 15-25% of total words

Never exceed 15% for any single phrase. Never drop below 5% overall.
